![Get access token](https://klaufel.com/tokens/img/get-access-token.png)

You can clone our Git repository:
`$ git clone git@github.com:klaufel/pattern-library-skeleton.git`

Install dependencies
`$ npm install`

And then, launch storybook
`$ npm run storybook`

Modify config with your figma API key and figma id
`$ figma.config.json`

![Create config](https://klaufel.com/tokens/img/create-config.png)

Generate design tokens as variables

`$ npm run tokens`

`$ npm run tokens:api` Generate tokens data base (only figma)

`$ npm run tokens:build` Generate all plattforms tokens vars with a figma data base
